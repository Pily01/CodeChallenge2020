Instrucciones

¡ Todos pueden programar !

  • Tema del Challenge

    El challenge consiste en utilizar la imagen de “Sati-Soda” como referencia para crear una aplicación / programa.

  • Integración de Cámara

    Sin importar el tipo de proyecto que se te haya ocurrido, tu programa debe de poder interactuar con una cámara de celular para escanear la imagen de SATI-Soda.

  • Objetivo

    Tu proyecto debe de ser capaz de escanear la imagen, reconocer la etiqueta, y realizar una acción. Un ejemplo es redirigir al usuario a una página web, video, animación, etc.

  • Sube Tu Proyecto a Nuestro Repositorio

    Sube tu proyecto en una carpeta con tus datos especificados en el README file del proyecto. (Da click en SUBIR PROYECTO para visitar el repositorio)

  • Límite 4 de Mayo 2020

    La idea más creativa de lo que sucederá al escanear la imagen será la ganadora del challenge. Aunque tu idea sea creativa tu código debe de funcionar para considerarte como ganador.
